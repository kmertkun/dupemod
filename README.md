<h2 id="english">🇬🇧 English</h2>

### 💎 Why Item Frame Duper?

**Item Frame Duper** is not just another utility; it's a streamlined tool designed for efficiency on anarchy servers. We've stripped away the bloat and focused purely on speed, reliability, and ease of use.

### 🔥 Key Features

| Feature | Description |
| :--- | :--- |
| **🚀 Blazing Fast** | Optimized `Speed Mode` ensures you get the maximum items per tick. |
| **🔄 Auto-Restock** | Never stop duping. The mod automatically refills frames and your hand. |
| **🛡️ Smart Protection** | Intelligent checks ensure you only hit what you're supposed to. |
| **⚙️ Fully Configurable** | Tweaks for every aspect: range, speed, tracers, X-Ray, and delays via Mod Menu. |

### 📖 Quick Start Guide

1.  **Install**: Drop the `.jar` into your mods folder. (Requires Fabric API).
2.  **Bind**: Go to controls and set your **Dupe Toggle** key.
3.  **Setup**: Place a frame around a wall element.
4.  **Execute**:
    *   Add Shulkers to your inventory.
    *   **Press the key.**
    *   *Profit.*

### ⚠️ Disclaimer

> [!WARNING]
> This mod is intended strictly for **ANARCHY** environments. Using this on regulated servers (SMP, Factions, etc.) is at your own risk and may result in a ban. We advocate for safe and responsible usage.

**Item Frame Duper** sıradan bir eklenti değildir; anarşi sunucularında maksimum verimlilik için tasarlanmış geliştirilmiş bir araçtır. Gereksiz özellikleri çıkardık ve tamamen hıza, güvenilirliğe ve kullanım kolaylığına odaklandık.

### 🔥 Ana Özellikler

| Özellik | Açıklama |
| :--- | :--- |
| **🚀 İnanılmaz Hızlı** | Optimize edilmiş `Hız Modu`, saniyede alabileceğiniz maksimum eşya sayısına ulaşmanızı sağlar. |

### 📖 Hızlı Başlangıç Rehberi

1.  **Kurulum**: İndirdiğiniz `.jar` dosyasını mods klasörünüze bırakın. (Fabric API gereklidir).
2.  **Atama**: Kontroller menüsüne gidin ve **Kopyalama (Dupe)** tuşunuzu belirleyin.
3.  **Hazırlık**: Duvara bir eşya çerçevesi yerleştirin.
4.  **Çalıştırma**:
